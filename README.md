## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Internet connection for Google Fonts

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mamun441998/mamun441998.github.io.git
   cd mamun441998.github.io
   ```

2. **Open in browser**
   - Open `index.html` in your web browser
   - Or use a local server for better development experience
